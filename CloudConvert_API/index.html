Project name:

Cloudconvert API

Documentation:

Цель проекта

Простое веб-приложение для практики работы с внешним API.

Позволяет загружать файлы и конвертировать их в другие форматы через CloudConvert Api

Основной функционал

-- Загрузка файла до 100 мб
-- Проверка размера и типа
-- Выбор формата конвертации
-- Отслеживание прогресса
-- Скачивание результата

Поддерживаемые форматы

Документы: PDF, DOC, DOCX
Изображения: JPG, PNG, GIF
Видео: MP4, AVI, MKV
Аудио: MP3, WAV, AAC
Архивы: ZIP, RAR, 7Z

Max file size - 100 MB
