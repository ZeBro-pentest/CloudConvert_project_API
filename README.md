# ☁️ CloudConvert_project_API

> Простой проект для практики работы с API. Позволяет загружать файлы и конвертировать их в другие форматы через CloudConvert. Проект на стадии разработки и предназначен для обучения работе с внешними API и созданию базового интерфейса. <br>

---

# **Документация** <br>

Цель проекта
Создать простое веб-приложение с использованием CloudConvert API, позволяющее пользователю конвертировать файлы в разные форматы через простой и интуитивный интерфейс. Проект предназначен для практики работы с API и базовой логикой фронтенда.

---
# **Об API** <br>

я использую бесплатную версию API от CloudConvert, и учитываю некоторые ограничения.

⚠️ Ограничения бесплатного плана: <br>
❌ 1 задание одновременно - если запустите второе, получите ошибку "too many jobs" <br>
❌ 25 минут в день - после использования нужно ждать до следующего дня <br>
❌ Файлы хранятся 24 часа - потом автоматически удаляются <br>
❌ Нет приоритета - ваши задания могут ждать в очереди <br>

**Основная логика работы:** <br>

CloudConvert на фронтенде работает через Job → Tasks: <br>

Import task – загружаем файл через upload (или через ссылку URL). <br>
Convert task – указываем, какой файл конвертировать и в какой формат. <br>
Export task – получаем ссылку на скачивание конвертированного файла. <br>

То есть структура всегда такая: <br>
**Upload → Convert → Export**
